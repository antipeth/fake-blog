安装fcitx5-rime输入法

paru -S fcitx5-im
paru -S fcitx5-material-color
paru -S fcitx5-rime
paru -S ruby

这里直接使用自动部署脚本。

git clone --depth=1 https://github.com/Mark24Code/rime-auto-deploy.git --branch latest
cd rime-auto-deploy
./installer.rb

输入3回车。

输入1回车。

然后在/etc/environment 文件末尾添加：

export GTK_IM_MODULE=fcitx5

export QT_IM_MODULE=fcitx5

export XMODIFIERS=@im=fcitx5

保存后退出，重启电脑之后，输入法就可以用了。

安装NVIDIA显卡驱动

paru -S nvidia-inst
nvidia-inst --drivers

需要等一会，不要着急。

这里我的series是545

nvidia-inst --series 545 -t

将--series参数后面的545改为你自己的series

例如以390为例，就是

 nvidia-inst --series 390 -t

将其中的这个：

COMMANDS TO RUN:
    pacman -Rs --noconfirm --noprogressbar --nodeps nvidia-dkms nvidia-hook nvidia-utils
    yay -Syu nvidia-390xx-dkms nvidia-390xx-utils nvidia-390xx-settings

里面的COMMANDS TO RUN给出的命令执行就可以安装显卡驱动。

pacman -Rs --noconfirm --noprogressbar --nodeps nvidia-dkms nvidia-hook nvidia-utils
yay -Syu nvidia-390xx-dkms nvidia-390xx-utils nvidia-390xx-settings

等待安装。

安装后执行：

nvidia-smi

来查看显卡驱动有没有正确安装。执行后有如下输出则证明显卡驱动安装成功。

vscodium更换官方插件源

找到vscodium的安装的文件夹。这里，我的eos的路径是/opt/vscodium-bin

在/opt/vscodium-bin/resources/app 中找到product.json 这个文件。

打开该文件，找到如下内容：

  "extensionsGallery": {
    "serviceUrl": "https://open-vsx.org/vscode/gallery",
    "itemUrl": "https://open-vsx.org/vscode/item"
  },

将其改为：

    "extensionsGallery": {
        "serviceUrl": "https://marketplace.visualstudio.com/_apis/public/gallery",
        "itemUrl": "https://marketplace.visualstudio.com/items"
    },

保存后重启vscodium，更换完毕。
